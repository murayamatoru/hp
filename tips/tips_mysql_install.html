2012年3月31日
■現象

MySQL5.5.22 のインストールが終了しない。

インストールできない。


■環境
MySQL 5.5.22 win32　（mysql-5.5.22-win32.msi） Windows7 Professional SP1 32bit
■現象の詳細
セットアップウィザード（MySQL Server 5.5 Setup）は完了するが、
次のインスタンス設定ウィザード（MySQL Server Instance Configuration Wizard）のExecuteボタンを押しても、
「Write configration file」の後、「応答なし（タスクマネージャ）」の状態が続く。

●Prepare configration
●Write configration file
○Start server
○Apply security settings

私は、この状態のままタスクマネージャで強制終了させた。

■原因
C:\ProgramData\MySQLが残っていたのが原因と思われる。（注：C:\Program Filesではない）
以前にMySQLをインストール／アンインストールしたことはないですか？
もしあれば、C:\ProgramData\MySQL が残っていませんか？

■対応
一度アンインストールし、C:\ProgramData\MySQLを削除後、再度インストールすると成功した。

■その他
アンインストール操作は「mysql-5.5.22-win32.msi」を起動した後の「Remove」で行った。
