Java Hello World 作成手順（Windows / IDEなし）

本手順書では、IDEを使わずにJavaプログラムを作成し、コンパイルして実行する方法を説明します。

① Java（JDK）をインストールする

Javaの開発には JDK（Java Development Kit） が必要です。

  1. ブラウザで以下の公式サイトにアクセスします。
    Oracle JDK ダウンロードページ
  2. Windows用の最新版（x64 Installer）をダウンロードします。
  3. ダウンロードしたファイル（.exe）を実行します。
  4. 画面の指示に従ってインストールします（基本は「次へ」でOK）。

■ インストール確認

  1. スタートメニュー → 「cmd」と入力 → 「コマンドプロンプト」を起動
  2. 次のコマンドを入力します。
java -version
javac -version

バージョン番号が表示されれば、インストール成功です。

② メモ帳などで HelloWorld.java を作る（必須ではないが　文字コード：UTF-8）

  1. デスクトップに新しいフォルダを作成します（例：JavaPractice）
  2. そのフォルダを開きます。
  3. 右クリック → 新規作成 → テキスト ドキュメント
  4. ファイル名を HelloWorld.java に変更します。
    （※ 拡張子が .txt にならないよう注意）
  5. そのファイルを右クリック → 「プログラムから開く」→「メモ帳」

■ 以下のプログラムを入力してください

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

入力後、保存してメモ帳を閉じます。

③ コマンドプロンプトでコンパイルする

  1. スタート → 「cmd」→ コマンドプロンプトを開く
  2. 作成したフォルダへ移動します。

例（デスクトップに作った場合）:

cd Desktop\JavaPractice

フォルダ内に HelloWorld.java があることを確認：

dir

コンパイルします：

javac HelloWorld.java

エラーが表示されなければ成功です。
同じフォルダ内に HelloWorld.class が生成されます。

④ Javaプログラムを実行する

次のコマンドを入力します：

java HelloWorld

■ 実行結果

Hello World

この表示が出れば成功です！

よくあるエラー

  • 'javac' は内部コマンドまたは外部コマンド〜
    → JDKが正しくインストールされていない、または環境変数PATHが設定されていない可能性があります。
  • class not found
    → フォルダの場所が間違っている可能性があります。

まとめ

  1. JDKをインストールする
  2. メモ帳などで HelloWorld.java を作る（必須ではないが　文字コード：UTF-8）
  3. javac でコンパイルする
  4. java で実行する

以上で、IDEを使わないJava Hello Worldの実行が完了です。
