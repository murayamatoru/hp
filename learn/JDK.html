 Java のバージョンについて

  • Java SE
    Java Standard Editionの略で、Java言語の標準仕様およびそのAPIを指します。Java言語やそのコアライブラリ、仮想マシン（JVM）などの仕様を定義しています。
  • OpenJDK
    Java SEのオープンソース実装であり、Javaの標準的な実装として広く利用されています。
  • Oracle JDK, Adoptium JDK, Temurin
    これらはOpenJDKをベースにした特定のベンダーによるJDKのディストリビューション（実装）です。それぞれ追加のツールやサポートが含まれる場合があります。
  • Java 8, Java 17
    JDKのバージョンを指し、Java言語やAPIの特定バージョンを意味します。
